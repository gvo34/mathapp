Endpoint shortcuts:

Alternative sevices calling, expecting input to be given as a string with the format q:array, where q is a positive integer value for the quantifier and array is a comma separated list of numbers.

Services:

  • /max for example: /max/3:2,3,5
  • /min for example /min/1:2,3,-1
  • /avg for example /avg/2,3,7,2,1,8
  • /median for example /median/2.4,3,6,1
  • /percentile for example /percentile/25:100,72,200,100,200
